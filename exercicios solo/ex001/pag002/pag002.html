Tipos de listas

Listas ordenadas

São Listas onde a ordem dos itens interferem e fazem diferença, para esse tipo de lista utilizamos a tag
<ol> e dentro desta, teremos a tag <li> que no caso são os itens da nossa lista;ex:
<ol type="">
|<li>Item nº1</li>
|<li>item nº2</li>
</ol>
O type="", que foi adicionado serve para definir o tipo de numeração dos itens da lista

  1. item nº1
  2. item nº2

Listas não ordenadas

São aqueles tipos de listas que a ordem dos iyens é irrelevante, como por exemplo uma lista de compras, esse tipo de listas se dá pela tag <ul>,
porém ainda continua a utilizar a tag <li> para os itens;ex:
<ul type="">
|<li>Item nº1</li>
|<li>item nº2</li>
</ul>
O type="", que foi adicionado serve para definir o tipo de pontuação dos itens da lista

  • item nº1
  • item nº2

Lista de definições

São Listas que servem para adicionar palavra que você pretende explicar / definir, esse tipo de lista se dá pela tag <dl>,
que acompanha as tags <dt>Termo</dt> e <dd>Definição</dd>; ex:
<dl>
|<dt>Casa</dt>
|<dd>moradia, lar, residencia etc...</dd>
</dl>

Casa
moradia, lar, residencia etc...

Links

Links internos

Dentro de links temos os limks internos, que são utilizados para a navegação dentro do seu próprio site, entre as suas páginas, para isso utilizamosa tag <a>
ex:
<a href="Local do arquivo da página" rel="next" target="_self"> nome do link </a>

  • href="" → Local em que o arquivo da prócima página está
  • rel="next" → Este irá indicar o tipo de link se sera de avanço ou retorno sendo: next de avanço e prev de retorno
  • target="_self" → Este indica se o arquivo é do próprio site ou de outro, para links internos iremos utilizar o _self

→ lembro que quando fazemos im link de retorno a página anterior caso ele esteja em outro pasta(que deve estar dentro da pasta principal do arquivo), utilizamos o parâmetro href do seguinte jeito:
href="../nome do arquivo"; sendo esse ../ utilizado para retornar uma pasta.

Links externos

Esse tipo de link serve para quendo quisermos redirecionar o usuário a outro site que não o nosso, ainda se utilizando da tag <a>, porém com alterações nos parâmetros
ex:
<a href="URL completa do site" target="_blank" rel="external"> nome do link </a>

  • href="" → URL completa do site que deseja ir.
  • rel="external" → Este irá indicar que o link será de um site exterior / diferente.
  • target="_blank" → Este indica se o arquivo é do próprio site ou de outro, para links externos iremos utilizar o _blank

Links para download

São links usados para fazer o download de um arquivo que foi anexado a esse link.
<a href="local do arquivo" download="nome completo do arquivo" type="aplicação para o tipo de arquivo"> nome do link </a>

  • href="local do arquivo" → selecionar o arquivo dentro da pasta.
  • download="nome completo do arquivo" → Neste deve ser colocado o nome completodo arquivo como está na salvo.
  • type="aplicação para o tipo de arquivo" → Deve ser colocado a aplicação para o tipo do arqui por exemplo "application/pdf", utiliado para arquivos pdf.
