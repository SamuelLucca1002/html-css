Cores

Parte teórica

Nos primeiros passos em CSS como de costume vimos primeiro algumas teorias.

Primeiramente vimos cores, tudo sobre elas em um site, tudo mesmo segue a listinha

  • Psicologia das cores

    • tudo sobre como as cores atraem as pessoas ou o que cada cor representa ou mostra de maneira subliminar e etc.

  • Representação das cores

    • Como as cores podem ser representadas dentro das CSS, sendo elas:

    • Pelo nome ex: blue
    • código hexadecimal ex: #0000ff
    • RGB ex: rgb(0, 0, 255)
    • HSL (hue, saturation, luminosity), ex: hsl(240, 100%, 50%)

  • harmonia de cores

    • Maneiras de se montar uma paleta de cores mais fácilmente, através de pré definições do próprio círculo etc...

Parte prática

Aprendemos a criar degrades de fundo para o site da seguinte forma:

body {
background-color: linear-gradient (to bottom, cor1, cor2, cor3;)
}
